# **Flash Message Demo Application**
## Description 
A rails 7 application created to learn how to utilize turbo stream flash messages. Created as a reference point, to be used whenever turbo flash messages are needed in other projects

## Installation
Fork repository, and open the project in your terminal. Install necessary gems and enter "rails s" in your console to begin the server
